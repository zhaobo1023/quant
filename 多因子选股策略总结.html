4 多因子评价框架

核心目标：科学评价因子有效性，回答"什么因子能预测未来收益"

主要功能

  • 因子计算：使用 TA-Lib 计算6个技术因子（动量、波动率、RSI、ADX等）
  • IC分析：计算因子值与未来收益的Spearman相关系数
  • 分层回测：按因子值分5组(Q1-Q5)，验证各组收益差异

核心代码结构

# 批量加载数据 all_data = batch_load_daily(start_date, end_date) # 计算技术因子 factor_df = calc_technical_factors(all_data) # IC分析 (因子值 vs 未来20日收益) ic_series, icir = calc_ic_series(all_data, factor_name, holding_days=20) # 分层回测 (5分位组) qr = quintile_backtest(all_data, factor_name, num_groups=5)

关键指标

指标说明判定标准
IC均值因子与收益的相关性>0.05 为有效因子
ICIRIC均值/IC标准差>0.5 优秀, >0.3 良好
Q5-Q1价差最高组与最低组收益差越大越好

6 小市值轮动策略

核心目标：利用A股"小市值效应"，结合动量+波动率过滤

策略对比

策略 选股逻辑 优点 缺点
纯小市值 只按市值排序，选最小的N只 简单直接 容易选到垃圾股
增强版 小市值 + 动量过滤 + 波动率过滤 排除暴跌股、极端波动股 参数较多

增强版过滤条件

# 过滤: 排除暴跌股 if momentum_20d < -20: continue # 过滤: 排除极端波动股 if volatility > 0.05: continue # 打分公式 score = (1 - cap_rank) * 0.5 # 小市值权重 + min(m20, 30) / 30 * 0.3 # 动量权重 + (1 - min(vol, 0.05) / 0.05) * 0.2 # 低波动权重

🎯 关键发现

  • 因子评价是基础：IC分析和分层回测是验证因子有效性的科学方法
  • 多因子组合更稳健：单因子容易被市场风格切换击败，多因子分散风险
  • 小市值效应在A股长期有效：但需要动量+波动率过滤排除垃圾股
  • ML能发现非线性关系：交互特征(动量*波动率)往往比单因子更有效
  • 滚动训练避免过拟合：Walk-Forward方式确保不使用未来信息
  • 选股+网格融合：因子选出"适合网格"的标的，缠论确定网格区间
